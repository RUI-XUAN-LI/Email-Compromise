# Tracking the Shadows: A Longitudinal, End-to-End Analysis of the Criminal Ecosystem for Email Account Compromise

### Email Compromise Tracking System

We provide the code for the email compromise tracking system in the `tracking_system` folder.

### Attacker IP Addresses

We provide the IP addresses of the attackers involved in the three phases of email compromise in the `attacker_ip` folder. The data is currently sampled, and the full version will be provided in the future.

### Liveness Probe Emails

We provide some liveness probe emails in the `probe_email` folder and hide sensitive information, such as email addresses, user IP addresses, passwords, etc.
